| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take security very seriously. If you have discovered a security vulnerability in VectorGraph-Engine, please report it to us responsibly. 

**Do NOT report security vulnerabilities through public GitHub issues.**

Instead, please send an email to the repository maintainer at **tusharsaini.in@gmail.com**. 

Please include the following in your report:
* Type of issue (e.g., SQL injection, Cross-Site Scripting, etc.)
* Full paths of source file(s) related to the manifestation of the issue
* The location of the affected source code (tag/branch/commit or direct URL)
* Any special configuration required to reproduce the issue
* Step-by-step instructions to reproduce the issue
* Proof-of-concept or exploit code (if possible)
* Impact of the issue, including how an attacker might exploit the issue

This project is maintained by a volunteer team, and depending on time of day and day of week, you can expect a response within a few days. We will review the vulnerability and, if confirmed, work on a patch to be released as soon as possible.
